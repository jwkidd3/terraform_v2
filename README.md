# Terraform Infrastructure as Code Mastery

**A comprehensive 3-day enterprise Terraform training course with specialized Terraform Cloud focus**

[![Terraform](https://img.shields.io/badge/Terraform-%3E%3D%201.9-623CE4)](https://terraform.io)
[![AWS](https://img.shields.io/badge/AWS-Cloud9-FF9900)](https://aws.amazon.com/cloud9/)
[![Duration](https://img.shields.io/badge/Duration-3%20Days-blue)](#)

## Course Overview

**Duration**: 3 days (9 hours hands-on labs + theory)
**Format**: 70% Hands-on / 30% Theory
**Level**: Introductory to Advanced
**Environment**: AWS Cloud9

## Course Structure

### **Day 1: Terraform Fundamentals** (5 Labs)
- **Lab 1**: Introduction to Terraform with Docker (45 min)
- **Lab 2**: AWS Infrastructure with Terraform (45 min)
- **Lab 3**: Advanced Variables and Configuration (45 min)
- **Lab 4**: Resource Dependencies and Lifecycle Management (45 min)
- **Lab 5**: Creating and Using Terraform Modules (45 min)

### **Day 2: Configuration and State** (4 Labs)
- **Lab 6**: Local State Management and Best Practices (45 min)
- **Lab 7**: Working with Terraform Registry Modules (45 min)
- **Lab 8**: Multi-Environment Deployment Patterns (45 min)
- **Lab 9**: VPC Networking and 2-Tier Architecture (45 min)

### **Day 3: Terraform Cloud** (3 Labs)
- **Lab 10**: Terraform Cloud Integration and Remote Execution (45 min)
- **Lab 11**: Terraform Cloud Workspaces (45 min)
- **Lab 12**: VCS-Driven Terraform Cloud with GitHub Integration (45 min)

## Getting Started

### For Instructors
1. Review presentations in [`presentations/`](presentations/)
2. Review labs in [`lab-exercises/`](lab-exercises/)

### For Students
1. Begin with [Lab 1 - Introduction to Terraform with Docker](lab-exercises/lab01/README.md)
2. Complete labs 1-12 in order
3. Prerequisites: AWS Cloud9 environment, Terraform Cloud account (free tier), GitHub account

### Prerequisites
- **AWS Cloud9** environment with appropriate IAM permissions
- **Terraform Cloud** account (free tier sufficient)
- **GitHub** account (for Labs 10-12)

## Learning Outcomes

### Terraform Core Skills
- Write production-ready HCL configurations
- Manage infrastructure state effectively
- Create reusable modules
- Implement dependency management and lifecycle rules

### Terraform Cloud Skills
- Configure remote execution and state management
- Manage multi-environment workspaces
- Implement VCS-driven GitOps workflows

### AWS Infrastructure
- Deploy VPC architectures with public/private subnets
- Configure EC2, S3, security groups, and load balancers
- Apply tagging strategies and security best practices

## Course Metrics

- **540 minutes** of hands-on Terraform experience (12 labs x 45 min)
- **12 infrastructure deployments** across Docker, AWS, and Terraform Cloud
- **Introductory to advanced** progression over 3 days
- **HashiCorp Terraform Associate** certification alignment
